## Cómo firmar

Este proyecto usa un flujo de firma automatizado (CLA Assistant Lite). Al abrir tu primer Pull
Request, un bot comentará solicitando tu firma. Para firmar, deja un comentario en el PR con el
texto exacto:

> **I have read the CLA Document and I hereby sign the CLA**

Tu firma (tu usuario de GitHub y el número de PR) se registra en el repositorio central
[`StrangeDaysTech/cla-registry`](https://github.com/StrangeDaysTech/cla-registry), en la carpeta
`weft/`, que es la fuente de verdad verificable de las firmas de este proyecto. El comentario que
dejas en tu PR es la evidencia original de tu firma (ver la sección 8); el registro central apunta a
él y congela la versión del CLA que firmaste. Solo necesitas firmar una vez; futuras contribuciones
quedan cubiertas mientras el contenido de este CLA no cambie. Si el CLA se actualiza a una nueva
versión, se te solicitará firmar de nuevo la versión vigente.
